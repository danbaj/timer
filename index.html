What does the button do?

The button is a representation of a physical button. PICTURE Both the physical button and this button have audio as their main output. This website is designed to showcase the functionality and provide information about the physical button.

The button can take two different input commands, clicking or holding the button.

The button starts a countdown timer and adds 1 minute for each press of the button. The timer can be reset by holding the button for 5 seconds. The timer was designed to provide an alternative to phone based timers that often requires several clicks on a touch screen in order start a countdown. The timer can be especially useful for practise sessions or timed exercises. YOUTUBE, GITHUB, ARDUINO, MORE
